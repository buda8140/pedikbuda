# LED Glow - Premium Control for RGB Strips

A professional, high-performance iOS application for controlling ELK-BLEDDM / Lotus Lantern LED hardware via Bluetooth Low Energy (BLE).

## Features
- **Premium UI**: Mesh gradients, glassmorphism, and neon aesthetics.
- **Live Sync**: Real-time 1024-bin FFT audio analysis for music-reactive effects.
- **Scheduling**: 15-minute gradual sunrise/sunset ramps via background tasks.
- **QR Setup**: Instant scan-to-connect for hardware identification.
- **Diagnostics**: Built-in console for protocol verification.

## Tech Stack
- **SwiftUI**: Modern declarative UI.
- **CoreBluetooth**: Low-level BLE stack.
- **Accelerate & vDSP**: High-performance digital signal processing.
- **BGTaskScheduler**: Background execution for timers.
